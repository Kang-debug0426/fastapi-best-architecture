- [x] 登录时写 Redis session（连接本地 Redis）
- [x] 认证时验证 Redis session
- [x] 加 DELETE `/auth/logout`，删 Redis key
- [x] 验证：logout 后同一 token 返回 401

### Task 4-3：FBA 对照
- [x] 去看 `backend/common/security/jwt.py`，找 JWT 生成和解码
- [x] 对比：FBA 的认证链比你的多了什么？

**完成标准**：能实现登录、认证、踢下线，能说出 JWT 和 Redis Session 各自的职责。✅

**踩坑记录**：
- passlib + bcrypt 不兼容，改用 bcrypt 直接调用 hashpw/checkpw
- verify_token 里误写成 r.delete 而不是 r.exists，导致每次访问都删 session
- Redis key 格式不一致：存时少了 username，查时找不到
- logout 函数里忘了取 username，key 拼错删不掉
